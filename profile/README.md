## What you'll find here

**Opinions on building with AI.** Sharp takes on what actually works in production. Accountability, memory, scoping, agent teams, observability. Read one and you'll have a clearer mental model for the architectural decision in front of you tomorrow.

**Practical tutorials.** Step-by-step builds for specific agent capabilities. Runnable code, the failure mode you'll hit if you skip the verification layer, and the fix. Finish one and you have a working agent doing the task.

**Monthly syntheses.** The connective tissue between everything that happened in agent-building this month. Long-form essays for when you want one piece that pulls the threads together instead of fifteen tabs.

**The Agent Index.** A vetted shortlist of tools an AI agent can call end-to-end. The bar is strict — if a step needs KYC, dashboard configuration, a sales call, or any click-to-enable gate, the tool doesn't make the list.

Both projects accept community contributions through pull requests. Each repo's `CONTRIBUTING.md` explains the bar.

## Why this exists

Most AI content is hype, demos, or vendor pitches dressed up as analysis. The Builder Weekly is for people building real systems with agents — the kind that have to work in production, not just impress on Twitter. We cover the patterns that survive a year of users.
